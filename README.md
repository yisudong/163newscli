## 功能特性

- 🔥 **热搜榜** — 实时 100 条热搜，含热度排名
- 📰 **要闻** — 网易首页头条
- 🎬 **娱乐** — 明星八卦，影视动态
- ⚽ **体育** — 赛事资讯，运动热点
- 💰 **财经** — 股市行情，商业新闻
- 🚗 **汽车** — 新车发布，购车指南
- 📖 **文章全文** — 直接在终端阅读完整正文

## 安装

```bash
npm install -g 163news-cli
```

或本地运行：

```bash
git clone https://github.com/yisudong/163newscli.git
cd 163newscli
npm install
npm run build
node dist/cli.js
```

## 使用

```bash
163news
```

### 键盘操作

| 按键 | 功能 |
|------|------|
| `↑` / `↓` | 上下移动光标 |
| `Enter` | 进入 / 阅读文章 |
| `q` / `Esc` | 返回上一级 |
| `j` / `k` | 向下 / 向上滚动文章 |

## 技术栈

- **Node.js** + **TypeScript**
- **Ink** — React TUI 框架，构建终端 UI
- **Axios** — HTTP 请求
- **node-html-parser** — HTML 解析

## 数据来源

抓取 `m.163.com` 页面中的 `window.__INITIAL_STATE__` 状态数据，文章详情来自 `c.m.163.com` JSON 接口，均为公开可访问的接口。
